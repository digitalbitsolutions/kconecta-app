# Module Map

## Property Module

- Handles all operations related to properties
- Provides endpoints for listing, detailing, creating, updating, and deleting properties

## Service Module

- Handles all operations related to services
- Provides endpoints for listing, detailing, creating, updating, and deleting services

## Admin Module

- Handles all operations related to admin users
- Provides endpoints for listing, detailing, creating, updating, and deleting admin users

# Backward Compatibility

- All changes are backward compatible with the existing system
- New endpoints are added to the existing modules
- Existing endpoints are updated to support new features
